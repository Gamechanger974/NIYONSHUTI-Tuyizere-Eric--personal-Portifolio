# Frontend Development CAT - Community E-Commerce Solution

##  Project Overview
This project is a web-based application designed to solve the challenge of digital access to essential goods in the **Kibungo Sector, Ngoma District**. It serves as a bridge between local vendors and residents, providing a streamlined shopping experience.

This project was developed as part of the **Learning Unit 1: Design GUI** Practical CAT.

##   Features
- **Responsive GUI:** A user-friendly interface designed first in Adobe XD and implemented using HTML5 and CSS3.
- **Dynamic Shopping Cart:** Functional product management using JavaScript (`shop.js`).
- **Secure Authentication:** User login and signup forms with validation logic (`auth.js`) to ensure data integrity.
- **Bootstrap Integration:** Styled with Bootstrap components for layout scalability and modern UI elements.
- **Social Media Connectivity:** Integrated links to Facebook and Twitter for community engagement.

##   Tech Stack
- **Languages:** HTML5, CSS3, JavaScript (ES6)
- **Frameworks:** Bootstrap 5
- **Design Tools:** Adobe XD (Prototypes)
- **Deployment:** Vercel

##   Project Structure
- `index.html`: The main landing page showcasing the community mission.
- `products.html` & `shop.js`: The core e-commerce functionality.
- `login.html` & `auth.js`: User authentication and form validation.
- `checkout.html`: Order processing with form input requirements.
